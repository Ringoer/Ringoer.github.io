需求分析

惯例先行需求分析

  1. 可以选择标签页排列的方向
  2. 选中的标签页应当有下划线显示
  3. 切换选中时，下划线应当有动画效果
  4. 应当允许更换颜色

那么可以整理出以下参数表格

参数 含义 类型 可选值 默认值
direction 方向 string row / column row
selected 默认选中 string 子项的 name 必填
color 颜色 string 任意合法颜色值 #f3678e

通过为子项设置 name 属性，来指定默认值
