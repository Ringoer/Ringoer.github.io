  2. 实验环境
    • 本机Windows10
    • 虚拟机系统VirtualBox
    • 虚拟机1 Windows7
    • 虚拟机2 Windows7
    • C#
    • Visual Studio 2017

  4. 实验总结
    基于VirtualBox
    1. 配置两台虚拟机时，应选择相同的串口
    2. 端口模式选择“主机管道”，路径/地址格式为\\.\pipe\xxxx，其中xxxx可以任意填写，但要保证两台虚拟机一致
    3. 对于选项“连接至现有通道或套接字”，其中一台虚拟机不勾选，称为pc1，另一台勾选，称为pc2
    4. 启动时，应先启动pc1，再启动pc2，否则会报如下错误
    5. 软件层面通信前应确保两台虚拟机上的软件都已经成功启动服务，否则不能互联
