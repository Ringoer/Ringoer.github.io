安全组

默认是全开，这肯定不行，建议开以下端口

协议 端口 描述
ICMP - 支持 Ping 服务
ICMPv6 - 支持 Ping 服务
TCP 20 ftp 连接端口
TCP 21 ftp 数据端口
TCP 22 ssh 登录端口
TCP 80 网站默认端口
TCP 443 ssl 默认端口
TCP 3306 mysql 默认端口
TCP 3389 windows 登录端口
TCP 6379 redis 默认端口
TCP （任意端口段） 自己的服务想用的端口
- 其余端口 均拒绝
